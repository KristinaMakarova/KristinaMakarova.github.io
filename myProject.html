Html

Язык разметки для создания структуры веб-страницы и представления контента.

Css

язык, который используют для визуального оформления веб-сайтов. Он работает в связке с языком разметки HTML. CSS отвечает за то, как выглядят объекты на странице: их размер, цвет, фоновое изображение, степень прозрачности, расположение относительно других элементов, поведение при наведении курсора, визуальное изменение кнопок при нажатии и т. п.

Js
